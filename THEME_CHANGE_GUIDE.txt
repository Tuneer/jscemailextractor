===============================================================================
CREATING A CUSTOM THEME
===============================================================================

1. Find your brand colors (e.g., from your logo)

2. Choose your primary color:
   --primary-color: #YOUR_COLOR

3. Create a darker shade (for gradients):
   --primary-dark: #DARKER_SHADE
   
   Tool: Use https://colordesigner.io/gradient-generator

4. Create a lighter shade (for hover effects):
   --primary-light: #LIGHTER_SHADE

5. Update the gradient:
   --primary-gradient: linear-gradient(135deg, var(--primary-color) 0%, var(--primary-dark) 100%);

6. Update light backgrounds (use 5-10% opacity of your color):
   --primary-light-bg: #YOUR_COLOR_LIGHT
   --primary-lighter-bg: #YOUR_COLOR_LIGHTER
   --primary-lightest-bg: #YOUR_COLOR_LIGHTEST
   
   Tool: Use https://rgba-to-hex.com/

7. Update shadow colors (convert hex to rgba):
   --primary-shadow-light: rgba(R, G, B, 0.1);
   --primary-shadow-medium: rgba(R, G, B, 0.2);
   --primary-shadow-strong: rgba(R, G, B, 0.4);
   
   Tool: Use https://www.rapidtables.com/convert/color/hex-to-rgb.html

===============================================================================
REVERTING CHANGES
===============================================================================

If you want to go back to the original Orange theme:

1. Open: frontend/src/styles.css

2. Replace the :root section with:
   :root {
     --primary-color: #ff6b35;
     --primary-dark: #f7931e;
     --primary-light: #ff8555;
     --primary-gradient: linear-gradient(135deg, var(--primary-color) 0%, var(--primary-dark) 100%);
     --primary-light-bg: #fff5f0;
     --primary-lighter-bg: #ffe8d9;
     --primary-lightest-bg: #fff9f6;
     --primary-shadow-light: rgba(255, 107, 53, 0.1);
     --primary-shadow-medium: rgba(255, 107, 53, 0.2);
     --primary-shadow-strong: rgba(255, 107, 53, 0.4);
   }

3. Save and refresh your browser
